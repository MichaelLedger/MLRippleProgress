# MLRippleProgress

Rounded Ripple Shape Progress View, usually used in download progress animation.

使用 CAShapeLayer + CADisplayLink

圆形水波加载动画，简单流程消耗小

自定义空间大，可加载多组色值的水波纹

将MLRippleProgress文件夹拖入工程即可使用
