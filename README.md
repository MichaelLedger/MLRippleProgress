# MLRippleProgress

Rounded Ripple Shape Progress View, Using CAShapeLayer & CADisplayLink.

圆形水波加载动画，自定义空间大，可加载多组色值的水波纹

使用 CAShapeLayer + CADisplayLink，消耗小

将MLRippleProgress文件夹拖入工程即可使用
